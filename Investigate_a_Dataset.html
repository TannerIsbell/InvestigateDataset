Introduction

Dataset Description

This data set is a combination of data that comes from the FBI's National Instant Criminal Background Check System (NICS) and state level data from census.gov. The NCIS is used by guns shopts to ensure their customers aren't ineligible to make a purchase. The NICS data consists of the following columns: month, state, permit, permit_recheck, handgun, long_gun, other, multiple, admin. There is also a column for handgun, long_gun, and other for each of the following: prepawn_, redemption_, returned_, private_sale_, return_to_seller_. Finally there is a rentals_ column for each handgun and long_gun. The Census data consists of the columns: fact, fact_note, and one for each state. The fact column explains what classification was being measured for that row. The state columns help to identify which state each value applies to. The two tables are related with the state column in the NCIS table to the corresponding state column in the Census table.

# Function to extract the lastest year in a column name
def extract_latest_year(col_name):
    years = re.findall(r'\d{4}', col_name)
    return max(int(y) for y in years) if years else -1 # -1 for columns without a year

# Create (column, year) pairs for remaining columns

Conclusions

The five facts that appear to have the greatest correlation with an increase in the number of firearm background checks are: total manufacturers shipments, housing units, population extimates, veterans, and households. The five facts that appear to have the greatest correlation with a decrease in the number of firearm background checks are: total retail sales, median household income, per capita income in past 12 months, median selected monthly owner costs-without a mortgage, and median value of owner-occupied housing units.

This information could be further improved with access to year over year data for each of the stats so you can compare how changes in each fact contribute to changes in firearm background checks rather than relying on a single point without accounting for how the numbers may be changing. The analysis that we are able to run is limited by this factor and doesn't allow us to truly measure how the number of background checks is effected by changes with these other factors. We can see that the number of background checks being run is trending upwards each year but aren't able to determine if the various facts are increasing or decreasing each year.

The census data was also missing values for some of the states (West Virginia, Maine, Vermont, Alaska, Hawaii, Wyoming, and Michigan) on some of the facts which prevented us from being able to fully understand how those facts may impact the background checks further limiting the analysis we were able to perform.
